# Convertidor Cedears IOL 📈

Este proyecto es una herramienta diseñada para inversores que utilizan la plataforma **InvertirOnline (IOL)**. Su objetivo principal era permitir a los usuarios calcular el rendimiento real en dólares de sus operaciones financieras realizadas originalmente en pesos argentinos.

## ⚠️ Estado del Proyecto: DEPRECADO
Este repositorio se encuentra actualmente **deprecado** y ya no recibirá actualizaciones. 

**Razón:** Recientemente, la plataforma oficial de InvertirOnline (IOL) lanzó una actualización en su interfaz y sistema de reportes que ya incluye el cálculo automático del rendimiento en dólares de los CEDEARs. Por lo tanto, la funcionalidad que brindaba esta aplicación ahora está integrada de forma nativa en el bróker.

## Funcionalidades Originales
- **Cálculo de Rendimiento:** Conversión automática de rendimientos de ARS a USD basándose en cotizaciones en tiempo real (CCL/MEP).
- **Análisis Financiero:** Herramienta para visualizar la ganancia real de los activos frente a la volatilidad de la moneda local.

## Tecnologías Utilizadas
- **JavaScript**
- **REACTNATIVE**
- **Implementacion de un servicio que hace scraping de una pagina web con datos financieros y se consumen a traves de una API REST**
